peoplefone bietet einen Elastic SIP-Trunk für [fonio.ai](http://fonio.ai) an.

Der Dienst muss derzeit noch über den Support aktiviert werden. Bitte senden Sie dazu eine E-Mail an [support@peoplefone.ch](mailto:support@peoplefone.ch) mit den folgenden Informationen:

- peoplefone Kunden-Nr.
- Kontaktperson / Telefonnummer
- peoplefone SIP-LINIEN-ID
- Telefonnummer(n) für die Integration in [FONIO.AI](http://FONIO.AI) (Format +4356336…)

\
Die Aktivierung erfolgt innerhalb von fünf Arbeitstagen.

Weitere Informationen finden Sie hier: https://www.peoplefone.com/de-de/sip-trunk
